A list of prompts curated by myself. The strength/influence of a prompt has been tested with multiple renders and usually a female character using sampling method Euler A.

The strength of these prompts will also work as negative prompts, so if you want to exclude something, use the stronger prompts for greater affect

I am working on adding images to correspond with each prompt. Images are cherrypicked to show the best example of each prompt. All images were created with the following settings:

Model: Waifu Diffusion 1.3 | Prompts: woman, {prompt} | Negative Prompts: {My default list} | Sampling Method: Euler | Sampling Steps: 50

Your results will vary a lot from what I'm able to generate, and some prompts will influence an image differently depending on what other prompts you use.
